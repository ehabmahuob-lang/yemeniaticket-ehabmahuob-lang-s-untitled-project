YemeniaTicket Pre-Login (PRO V5)

✅ تعديلات حسب طلبك:
- حل تداخل "الخصوصية" مع "تسجيل الدخول" عبر إعادة تصميم الهيدر + تحديد أعمدة grid + تقليل padding لعناصر القائمة + تمرير أفقي عند الحاجة.
- تكبير الشعار، وضبط تناسق الاسم: اليمنية تكت || Yemenia Ticket
- تحويل "احجز أونلاين خلال دقائق" إلى "احجز أونلاين"
- إضافة ذكر التنبيهات الآلية عبر واتساب داخل النصوص.
- إضافة باص متحرك (عجلات + دخان) في الهيرو + لودر تحميل بالباص عند الضغط على الأزرار.
- إضافة صفحات جديدة:
  - vehicles.html (حجز سيارات)
  - flights.html (رحلات جوية قريبًا)

تشغيل محلي:
  python -m http.server 8000
ثم:
  http://localhost:8000


✅ تم حذف حركة الباص نهائيًا واستبدالها بلودر سبينر بسيط واحترافي.
